# 設計メーカー クラス図

対象は `src/` 配下の実装である。ブラウザのグローバルスコープで読み込まれるプレーン JavaScript 構成のため、`DiagramController` がアプリケーションの調整役となり、状態管理・入出力・描画・UI をそれぞれ専用のクラスへ委譲する。

```mermaid
classDiagram
  direction LR

  class DiagramController {
    -elements: Object
    -state: DiagramState
    -connectMode: boolean
    -connectSourceId: string
    -undoStack: DiagramState[]
    -pendingHistoryState: DiagramState
    +init()
    +addClass()
    +addGroup()
    +importMermaid(text)
    +alignSelectionToGrid()
    +zoom(request, center)
    +undo()
    +updateState(nextState, shouldSave, renderInspector, trackHistory)
    +render(renderInspector)
  }

  class DiagramModel {
    +createInitialState() DiagramState
    +normalizeState(input) DiagramState
    +createClass(input, state) ClassNode
    +createGroup(input) Group
    +createProperty(input) Property
    +createMethod(input) Method
    +createParameter(input) Parameter
    +createRelationship(input) Relationship
    +addClass(state, input) DiagramState
    +updateClass(state, classId, patch) DiagramState
    +removeClass(state, classId) DiagramState
    +addGroup(state, input) DiagramState
    +removeGroup(state, groupId) DiagramState
    +addRelationship(state, input) DiagramState
    +updateRelationship(state, relationshipId, patch) DiagramState
    +reverseRelationship(state, relationshipId) DiagramState
    +snapClassesToGrid(state, classIds, gridSize) DiagramState
    +validate(state) ValidationResult
  }

  class MermaidParser {
    -model: DiagramModel
    +parse(text) DiagramState
  }

  class MermaidSerializer {
    +serialize(state) string
  }

  class LayoutService {
    +applyInitialLayout(state) DiagramState
  }

  class StorageService {
    -storageKey: string
    +load() LoadResult
    +save(state) SaveResult
    +clear()
  }

  class AnalyticsService {
    -endpointUrl: string
    -clientIdKey: string
    -sentDateKey: string
    -timeZone: string
    +trackPageView()
    +getClientId() string
    +createClientId() string
    +getToday() string
  }

  class SvgRenderer {
    -svg: SVGElement
    -handlers: Object
    -viewportGroup: SVGGElement
    -drag: Object
    -state: DiagramState
    +bindEvents(handlers)
    +setConnectSource(classId)
    +render(state, options)
    +getGroupBounds(state) Map
    +relationshipPath(source, target) Object
  }

  class InspectorView {
    -root: HTMLElement
    -handlers: Object
    -isCollapsed: boolean
    -lastState: DiagramState
    +bindEvents(handlers)
    +render(state)
    +renderClass(classNode)
    +renderGroup(group, classes)
    +renderRelationship(relationship, classes)
  }

  class ModalView {
    -root: HTMLElement
    -handlers: Object
    +bindEvents(handlers)
    +close()
    +showImport()
    +showExport(text)
    +showError(message)
    +confirm(message, onConfirm)
  }

  class DiagramState {
    +version: number
    +classes: ClassNode[]
    +groups: Group[]
    +relationships: Relationship[]
    +viewport: Viewport
    +selection: Selection
  }
  <<value object>> DiagramState

  class ClassNode {
    +id: string
    +name: string
    +kind: string
    +stereotype: string
    +visibility: string
    +isAbstract: boolean
    +position: Point
    +size: Size
    +properties: Property[]
    +methods: Method[]
  }
  <<value object>> ClassNode

  class Group {
    +id: string
    +name: string
    +label: string
    +parentId: string
    +classIds: string[]
  }
  <<value object>> Group

  class Relationship {
    +id: string
    +sourceClassId: string
    +targetClassId: string
    +type: string
    +label: string
    +sourceMultiplicity: string
    +targetMultiplicity: string
  }
  <<value object>> Relationship

  class Property {
    +id: string
    +name: string
    +type: string
    +visibility: string
    +isStatic: boolean
    +isReadonly: boolean
    +isAbstract: boolean
    +defaultValue: string
    +description: string
  }
  <<value object>> Property

  class Method {
    +id: string
    +name: string
    +returnType: string
    +visibility: string
    +isStatic: boolean
    +isAbstract: boolean
    +isAsync: boolean
    +parameters: Parameter[]
    +description: string
  }
  <<value object>> Method

  class Parameter {
    +id: string
    +name: string
    +type: string
    +defaultValue: string
    +isOptional: boolean
  }
  <<value object>> Parameter

  DiagramController *-- DiagramModel
  DiagramController *-- StorageService
  DiagramController *-- MermaidParser
  DiagramController *-- MermaidSerializer
  DiagramController *-- LayoutService
  DiagramController *-- SvgRenderer
  DiagramController *-- InspectorView
  DiagramController *-- ModalView
  MermaidParser *-- DiagramModel
  DiagramController --> DiagramState : 管理
  MermaidSerializer --> DiagramState : 変換
  LayoutService --> DiagramState : 配置
  StorageService --> DiagramState : 保存/復元
  SvgRenderer --> DiagramState : 描画
  InspectorView --> DiagramState : 編集対象表示
  DiagramState *-- "0..*" ClassNode
  DiagramState *-- "0..*" Group
  DiagramState *-- "0..*" Relationship
  ClassNode *-- "0..*" Property
  ClassNode *-- "0..*" Method
  Method *-- "0..*" Parameter
  Group "0..*" --> "0..*" ClassNode : classIds
  Group "0..*" --> "0..1" Group : parentId
  Relationship "0..*" --> "1" ClassNode : sourceClassId
  Relationship "0..*" --> "1" ClassNode : targetClassId
```

## 補足

- `DiagramModel` は状態を破壊的に変更せず、新しい `DiagramState` を返す。`DiagramController` が履歴、保存、再描画のタイミングを統制する。
- `MermaidParser` は独自に `DiagramModel` を保持し、Mermaid テキストを編集可能な状態へ変換する。位置情報がない入力には `LayoutService` が初期配置を適用する。
- `SvgRenderer`、`InspectorView`、`ModalView` はイベントハンドラーを `DiagramController` から受け取り、UI 操作をコントローラーへ戻す。
- `AnalyticsService` は `main.js` で別途生成され、ページ表示を日次単位で計測する。図の編集状態には関与しない。
