# クラスサイズ可変 作業手順書

この手順書では、クラスボックスの右端・下端・右下角をドラッグしてサイズ変更できるようにする。変更対象は以下の4ファイル。

- `src/models/diagramModel.js`
- `src/views/svgRenderer.js`
- `src/controllers/diagramController.js`
- `src/styles.css`

## 1. `src/models/diagramModel.js` を変更する

### 1-1. 最小サイズ定数を追加する

ファイル先頭を次のように変更する。

```javascript

  width: 180,
  height: 110

function createId(prefix) {

  return `${prefix}_${nextId.toString(36)}`;
```

`normalizeState()` 内のクラス正規化処理で、以下の行を探す。

### 1-3. `createClass()` の `size` を正規化する

`createClass()` 内の以下の行を探す。

```javascript

```

次のように置き換える。

### 1-4. `normalizeClassSize()` を追加する

`createClass()` の直後、`createProperty()` の前に次のメソッドを追加する。


### 2-12. `centerOf()` と `edgePoint()` を変更する

ファイル下部の `centerOf()` と `edgePoint()` を探す。

```javascript
function centerOf(classNode) {
  return {
    x: classNode.position.x + classNode.size.width / 2,
    y: classNode.position.y + classNode.size.height / 2
  };
}

function edgePoint(classNode, toward) {
  const center = centerOf(classNode);
  const dx = toward.x - center.x;
  const dy = toward.y - center.y;
  const halfWidth = classNode.size.width / 2;
  const halfHeight = classNode.size.height / 2;
  const scale = Math.max(Math.abs(dx) / halfWidth, Math.abs(dy) / halfHeight, 1);
  return {
    x: center.x + dx / scale,
    y: center.y + dy / scale
  };
}
```

次のように置き換える。

```javascript
function centerOf(classNode, size) {
  return {
    x: classNode.position.x + size.width / 2,
    y: classNode.position.y + size.height / 2
  };
}

function edgePoint(classNode, size, toward) {
  const center = centerOf(classNode, size);
  const dx = toward.x - center.x;
  const dy = toward.y - center.y;
  const halfWidth = size.width / 2;
  const halfHeight = size.height / 2;
  const scale = Math.max(Math.abs(dx) / halfWidth, Math.abs(dy) / halfHeight, 1);
  return {
    x: center.x + dx / scale,
    y: center.y + dy / scale
  };
}
```

`updateClassPositions(positions)` の直後に次のメソッドを追加する。

```javascript

    this.updateState(this.model.updateClass(this.state, classId, { size }), false, false);
  }

  finishClassResize(classId, moved) {
    this.updateState(this.model.select(this.state, { type: "class", id: classId }), moved, true);
  }
```

## 5. 動作確認

1. `index.html` をブラウザで開く。
2. クラスをクリックし、右端・下端・右下角にハンドルが出ることを確認する。
3. 右端ハンドルをドラッグし、幅だけが変わることを確認する。
4. 下端ハンドルをドラッグし、高さだけが変わることを確認する。
5. 右下角ハンドルをドラッグし、幅と高さが同時に変わることを確認する。
6. 最小サイズより小さくしようとしても、幅180px・高さ110px未満にならないことを確認する。
7. 関係線があるクラスをリサイズし、線が新しい外周に追従することを確認する。
8. ページを再読み込みし、リサイズ後のサイズが復元されることを確認する。
9. Mermaid エクスポートを実行し、サイズ情報が出力されないことを確認する。
10. Mermaid インポート後のクラスもリサイズできることを確認する。
11. クラス移動、範囲選択、Alt/中ボタンパン、Ctrl+ホイールズーム、関係選択が従来通り動作することを確認する。
